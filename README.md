# AISvisualizer
It is a web application for visualization positions of marine units based on data from an AIS receiver. Development of application was divided into two parts: server-side application and client-side application. Database was designed as relational MS SQL database with use of T-SQL. However backend was realized as ASP .NET Core v3.1 Web API application with use of Entity Framework Core v3.1.2. Client-side application was developed as Angular v8 application, with use of TypeScript, HTML and CSS.
